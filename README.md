# RAG Chatbot

This is a **RAG (Retrieve-Augment-Generate) chatbot** built with Python, LangChain, FAISS, and OpenAI LLM.

## Features
- Upload PDF, DOCX, or image files
- Extract text from documents and images
- Split text into chunks and create FAISS embeddings
- Ask questions and get answers from documents
